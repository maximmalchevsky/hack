# Workie — система актуализации рабочего времени



## Что умеет

- **Дашборд под роль** — 6 ролей (сотрудник, руководитель, PM, HR, аналитик, админ),
  у каждой свой набор виджетов.
- **Профиль и рабочие часы** — график по дням недели, часовой пояс, исключения
  (отпуск/больничный/командировка), история изменений (temporal-версии).
- **Карта команды** — heatmap доступности по часам и дням с учётом часовых поясов.
- **Загрузка** — занятые часы против рабочих, таймлайн встреч и задач.
- **Диагностика** — группировка сотрудников: устаревший график, перегрузка,
  встречи вне графика, конфликт часовых поясов, расхождение с HR.
- **Конфликты** — пересечения встреч и события вне рабочих часов.
- **Рекомендации** — объяснимые подсказки (что и почему) с кнопкой действия.
- **Планировщик** — поиск окна для встречи (топ-слоты), ручное создание встречи
  в любую дату, анти-burnout защита (soft-block при >35ч встреч в неделю).
- **Дорожная карта HR** — приоритезированный список «у кого попросить подтвердить график».
- **Аналитика** — динамика метрик, графики по командам.
- **ИИ-ассистент** — чат с объяснениями, ссылками на показатели и быстрыми вопросами.
- **Уведомления** — in-app + WebSocket в реальном времени, рассылки.
- **Интеграции** — Яндекс.Календарь (CalDAV/OAuth), iCal/ICS, Jira, Yandex Tracker.
- **Календарные инвайты** — `.ics` через SMTP (iMIP), автоприём RSVP через IMAP.
- **Задачи** — таск-трекер с оценкой времени и планом.

## Метрики

Для каждого сотрудника считаются (пакет `internal/analytics`):

| Метрика | Что означает |
|---|---|
| **A** — актуальность | насколько свежий график: `A = 1 − d/D`, `D = 90` дней |
| **C** — конфликты | доля встреч вне графика + двойные брони (overlap) |
| **L** — загрузка | занятые часы / рабочие часы за неделю |
| **Z** — TZ-drift | доля событий со смещением от заявленного часового пояса |
| **H** — HR-mismatch | расхождение фактической активности с HR-форматом |
| **R** — интегральный риск | `R = w1(1−A) + w2·C + w3·L + w4·Z + w5·H` (веса в env) |

## Стек

| Слой | Технологии |
|---|---|
| Backend | Go 1.26, Fiber v3, pgx v5, Asynq (очереди), zerolog |
| БД / кэш | PostgreSQL 16, Redis 7 |
| Миграции | golang-migrate (встроены через `embed.FS`) |
| Frontend | SvelteKit 2 + Svelte 5, TailwindCSS, ECharts, Tabler Icons |
| ИИ | GigaChat (Sber) + rule-based fallback |
| Auth | JWT (access/refresh), bcrypt, AES-256-GCM для токенов |
| Реалтайм | WebSocket + Redis pub/sub |
| Прокси | Caddy 2 (TLS, прод) |
| Деплой | Docker Compose |
